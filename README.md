# banco-pix
Simulação de transferência via pix entre contas bancárias.

**Telas**
* Login
* Cadastro
* Home
* Perfil
* Fazer transferência
* Extrato

## Tecnologias usuadas
**Front-end**
* React Native
* Styled-components

**Back-end**
* C#
* .NET
* API criada de forma simples, sem uso de frameworks, e simulação de banco de dados com arquivo JSON.
